PHP Manual Language Switcher

Redirects the PHP Manual page to a specified PHP Manual language page.

Language Setting.
  • (en) Select the language you want to redirect to.
  • (pt_BR) Selecione o idioma para o qual deseja redirecionar.
  • (zh) 选择您要重定向到的语言。
  • (fr) Sélectionnez la langue vers laquelle vous souhaitez rediriger.
  • (de) Vælg det sprog, du vil omdirigere til.
  • (ja) リダイレクトしたい言語を選択してください。
  • (ro) Selectați limba către care doriți să redirecționați.
  • (ru) Выберите язык, на который хотите выполнить перенаправление.
  • (es) Seleccione el idioma al que desea redirigir.
  • (tr) Yönlendirmek istediğiniz dili seçin.
